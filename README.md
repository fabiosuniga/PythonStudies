# 🐍 Jornada de Estudos em Python

Bem-vindo ao meu repositório de estudos! Este espaço foi criado para documentar a minha evolução contínua na linguagem **Python**. Aqui, armazeno os exercícios, desafios e projetos práticos que desenvolvo ao longo do meu aprendizado.

O foco principal deste repositório não é apenas criar códigos que funcionem, mas sim aprender a construir soluções de forma limpa, eficiente e utilizando boas práticas de mercado (o famoso *Pythonic Way*).

---

## 📚 O Que Você Vai Encontrar Aqui

Durante os estudos documentados nestes arquivos, explorei desde os fundamentos lógicos até conceitos arquiteturais da linguagem. Os scripts cobrem os seguintes temas:

* **Estruturas de Dados:** Uso inteligente de Listas, Tuplas, Dicionários e Conjuntos (Sets).
* **Otimização e Lógica:** Aplicação de *List Comprehensions* e expressões geradoras para reduzir a verbosidade do código.
* **Manipulação de Textos:** Técnicas de fatiamento e limpeza de strings extraídas de logs e dados brutos.
* **Código Resiliente:** Tratamento de erros e exceções (blocos `try / except`) para evitar o travamento de aplicações.
* **Programação Orientada a Objetos (POO):** Criação de classes, uso de construtores, métodos de instância e interação dinâmica entre objetos.

---

## 🚀 Habilidades e Técnicas Aplicadas

Para elevar o nível da escrita do código, os exercícios focam na aplicação dos seguintes conceitos:

* Desempacotamento de variáveis (*Unpacking*) em múltiplas atribuições simultâneas.
* Construção e agrupamento de dicionários de forma segura usando o método `.get()`.
* Validações complexas resolvidas em única linha com as funções nativas `any()` e `all()`.
* Substituição de loops demorados por operações matemáticas de conjuntos (Intersecção, Diferença, União).
* Tratamento de exceções específicas (ex: `ValueError`), garantindo a segurança de sistemas que lidam com dados imprevisíveis.
